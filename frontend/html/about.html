My Mission

This is an academic project as part of a Full Stack Python Development course that was delivered in person at John Bryce School in Tel Aviv, Israel. The software was developed using various tools and technologies learned in the course, primarily using Flask as the backend framework, alongside Python, JavaScript, and HTML, among others. The software was originally developed from scratch without using any external code.

The software includes all stages of borrowing in a standard library using CRUD (create, read, update, and delete) operations and a database that comprises dozens of customers and dozens of books. Instead of permanent deletion, the "delete" operation is implemented as a soft delete, temporarily deactivating or disabling records rather than removing them permanently. The library page caters to three audiences: librarians, customers, and developers – each with unique usernames and passwords.
